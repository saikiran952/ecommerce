# 🛍️ E-Commerce Website Project

## 📌 About the Project

This is a complete e-commerce website built using **HTML, CSS, PHP**, and **MySQL**, with some JavaScript functionality for interactivity. It includes user and admin login systems, product management (add, edit, delete), image uploads, and social login integration via **Google** and **Facebook** OAuth.

## 🚀 Features

* User registration and login with session handling
* Admin login with access to dashboard and product control
* Secure password hashing using `password_hash()`
* Product image upload and display
* Google and Facebook social login (OAuth2)
* Fully responsive and modern UI with background video
* Dynamic product listings connected to MySQL database

## 🛠️ Technologies Used

* **Frontend**: HTML5, CSS3, JavaScript
* **Backend**: PHP
* **Database**: MySQL (phpMyAdmin)
* **OAuth**: Google & Facebook via Composer (league/oauth2-client)

## ⚙️ How to Run

1. Clone this repository
2. Set up a local server using XAMPP or WAMP
3. Import the SQL database using phpMyAdmin
4. Run `composer install` to load dependencies (OAuth)
5. Update your Google/Facebook app credentials
6. Visit `http://localhost/ecommerce` in your browser
